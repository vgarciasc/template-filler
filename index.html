INSTRUCTIONS: upload a template and create areas with the mouse. Click in an area to select it. Once selected, upload an Image by URL or by File. You will be requested to crop the image, and then the area will be filled with the cropped image. When you are satisfied, exit edit mode, right-click the canvas and save the picture.

UNDO (Z key): undo the last action.
WIPE (W key): resets the entire canvas.
DUPLICATE (D key): duplicates the selected area.
CLEAR (C key): deletes the image of the selected area.
REMOVE (R key): removes the selected area.
TOGGLE EDIT (T key): toggles the edit mode. Outside of exit mode, the canvas only contains the template and your images.
